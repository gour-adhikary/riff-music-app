# riff-music-app

A simple music streaming application built with Java and React.

## Tech Stack
- **Backend**: Java (Spring Boot)
- **Frontend**: React (Vite)

## Architecture
- **Communication**: REST API for fetching song metadata and streaming audio.
- **State Management**: Playback progress and state are managed entirely on the frontend.

## Getting Started

### Prerequisites
- Java 17+
- Node.js & npm

### Structure
- `/backend` - Spring Boot Server
- `/frontend` - React Application
